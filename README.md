# Rooster Wars Web3 Backend

## Pre Requisites
Before running any command, make sure to install dependencies:
```
$ yarn 
```

## Develop
Run TS node and watch for changes:
```
$ yarn dev
```

## Build
Build and run the program :
```
$ yarn build
$ yarn start
```

## Linting
Lint files: 
```
yarn lint:fix
```
